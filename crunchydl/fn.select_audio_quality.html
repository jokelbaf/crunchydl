Expand description

Select an audio representation by QualitySelection.

Audio is ranked by bandwidth, then sampling rate. Height-based policies map to the best available bandwidth.

§Errors

Returns SelectionError when there are no candidates, or when an exact bandwidth request cannot be met under FallbackPolicy::Exact.
