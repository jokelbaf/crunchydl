pub fn canonicalize_episode_batch(
    episodes: impl IntoIterator<Item = ResolvedMedia>,
) -> Result<Vec<ResolvedMedia>, BatchError>
Expand description

Merge localized duplicate episode records into logical episodes.

Entries with the same service identifier are treated as one episode. When identifiers are absent, series id, logical season position, episode label, and sequence number form the fallback key. Audio versions and subtitle locales are unioned deterministically.

§Errors

Returns BatchError::NonEpisode if the input contains another media kind.
