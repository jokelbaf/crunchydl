Crate crunchydl 

Source
Expand description

Pure-Rust Crunchyroll downloader library.

crunchydl discovers Crunchyroll media, selects versions and tracks, downloads DASH media and subtitles, obtains content keys from a caller-supplied DRM device, decrypts fragmented MP4 media, and muxes a finished Matroska file - all without invoking any external executable.
