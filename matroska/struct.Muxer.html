pub fn write<W, I>( output: &mut W, tracks: &[Track], packets: I, chapters: &[Chapter], attachments: &[Attachment], options: &MuxOptions, ) -> Result<(), Error>
where W: Write + Seek, I: IntoIterator<Item = Packet>,

Write a complete seekable Matroska segment.

Packets must be ordered by nondecreasing decode time. Presentation time may differ, including for AVC B-frames.

§Errors

Returns an error for unsupported metadata, invalid ordering/timestamps, overflow, or output I/O failure.

pub fn write_fallible<W, I>( output: &mut W, tracks: &[Track], packets: I, chapters: &[Chapter], attachments: &[Attachment], options: &MuxOptions, ) -> Result<(), Error>
where W: Write + Seek, I: IntoIterator<Item = Result<Packet, Error>>,

Write from a fallible streaming packet source.

This variant lets a demuxer surface an input error without first collecting the complete media stream.

§Errors

Returns the first packet-source, validation, overflow, or I/O error.
